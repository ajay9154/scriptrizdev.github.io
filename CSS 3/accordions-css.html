Responsive Accordions Design in HTML CSS & JavaScript

By
wynkCode -
May 9, 2023

Accordions are a common UI pattern used to hide or reveal content in a collapsible manner. In this tutorial, we will learn how to create responsive accordions using HTML, CSS, and JavaScript. The accordions will adjust their layout based on the screen size to provide a seamless user experience on different devices.

Step 1: Set up the HTML structure:

  • Begin by creating the HTML structure for the accordions. Each accordion item will consist of a heading and a content section. Use appropriate HTML elements to represent the structure, such as <div> containers and <h2> for the headings.

Step 2: Style the accordions using CSS:

  • Apply CSS styles to create the desired appearance for the accordions. This includes defining the dimensions, colors, borders, and transitions. Use CSS selectors to target the accordion elements and apply the appropriate styles.

Step 3: Implement responsiveness:

  • To make the accordions responsive, use CSS media queries to adjust the layout based on the screen size. Define different styles for smaller screens to ensure the accordions are displayed optimally on mobile devices and tablets. 
javascript
const faq = document.querySelector('.faq');
faq.addEventListener('click', event => {
  const question = event.target.closest('.faq__question');
  if (!question) return;
  const answer = question.nextElementSibling;
  const currentAnswer = faq.querySelector('.faq__answer[aria-hidden="false"]');

  if (currentAnswer === answer) {
    answer.setAttribute('aria-hidden', 'true');
    answer.parentNode.classList.remove('faq__item--expanded');
    question.setAttribute('aria-expanded', 'false');
  } else {
    if (currentAnswer) {
      currentAnswer.setAttribute('aria-hidden', 'true');
      currentAnswer.parentNode.classList.remove('faq__item--expanded');
      currentAnswer.previousElementSibling.setAttribute('aria-expanded', 'false');
    }
    answer.setAttribute('aria-hidden', 'false');
    answer.parentNode.classList.add('faq__item--expanded');
    question.setAttribute('aria-expanded', 'true');
  }
});

  • Implement JavaScript functionality to enable the accordion behavior. This involves handling user interactions, such as clicks on the accordion headers, to expand or collapse the corresponding content sections. Use event listeners to capture the user actions and modify the DOM accordingly.

By following this tutorial, you have learned how to create responsive accordions using HTML, CSS, and JavaScript. You can now apply this knowledge to enhance the user experience on your website by adding collapsible content sections that adapt to different devices. Experiment with different styles and features to customize the accordions to fit your specific design requirements.
