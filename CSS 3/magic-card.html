Create a Responsive Side Navigation Menu Bar using HTML, CSS And JavaScript

By
wynkCode -
May 9, 2023

In this tutorial, we will create a magic-themed animated card using CSS. The card will have a gradient background and will rotate continuously, giving it a mystical appearance. We will also add hover effects to enhance the interactivity. Let's get started!

Step 1: Setting up the HTML Structure

First, let's set up the basic HTML structure for our magic card. Open your HTML file in a text editor or code editor and add the following code: 

html
<!DOCTYPE html>
                    <html>
                    <head>
                      <title>Magic Card with CSS</title>
                      <link rel="stylesheet" href="styles.css">
                    </head>
                    <body>
                      <div class="card">Magic Card</div>
                    </body>
                    </html>

Just like before, we have included a link to an external CSS file named styles.css, which we will create in the next step. The card is represented by a << /span>div> element with the class "card" and contains the text "Magic Card".

Step 2: Creating the CSS Styles

Now, let's create the CSS styles for our magic card. In the same directory as your HTML file, create a new file named styles.css and open it in your text editor or code editor. Add the following code: 

css
@property --rotate {
                    syntax: "<angle>";
                    initial-value: 132deg;
                    inherits: false;
                  }
                  
                  :root {
                    --card-height: 65vh;
                    --card-width: calc(var(--card-height) / 1.5);
                  }
                  
                  body {
                    min-height: 100vh;
                    background: #212534;
                    display: flex;
                    align-items: center;
                    flex-direction: column;
                    padding-top: 2rem;
                    padding-bottom: 2rem;
                    box-sizing: border-box;
                    font-family: "Poppins", sans-serif;
                  }
                  
                  .card {
                    background: #191c29;
                    width: var(--card-width);
                    height: var(--card-height);
                    padding: 3px;
                    position: relative;
                    border-radius: 6px;
                    justify-content: center;
                    align-items: center;
                    text-align: center;
                    display: flex;
                    font-size: 1.5em;
                    color: rgb(88, 199, 250, 0%);
                    cursor: pointer;
                  }
                  
                  .card:hover {
                    color: rgb(88, 199, 250, 100%);
                    transition: color 1s;
                  }
                  
                  .card:hover:before,
                  .card:hover:after {
                    animation: none;
                    opacity: 0;
                  }
                  
                  .card::before {
                    content: "";
                    width: 104%;
                    height: 102%;
                    border-radius: 8px;
                    background-image: linear-gradient(var(--rotate), #5ddcff, #3c67e3 43%, #4e00c2);
                    position: absolute;
                    z-index: -1;
                    top: -1%;
                    left: -2%;
                    animation: spin 2.5s linear infinite;
                  }
                  
                  .card::after {
                    position: absolute;
                    content: "";
                    top: calc(var(--card-height) / 6);
                    left: 0;
                    right: 0;
                    z-index: -1;
                    height: 100%;
                    width: 100%;
                    margin: 0 auto;
                    transform: scale(0.8);
                    filter: blur(calc(var(--card-height) / 6));
                    background-image: linear-gradient(var(--rotate), #5ddcff, #3c67e3 43%, #4e00c2);
                    opacity: 1;
                    transition: opacity 0.5s;
                    animation: spin 2.5s linear infinite;
                  }
                  
                  @keyframes spin {
                    0% {
                      --rotate: 0deg;
                    }
                    100% {
                      --rotate: 360deg;
                    }
                  }

Step 3: Previewing the Magic Card

Save the styles.css file and refresh your HTML file in a web browser. You should now see the magic-themed card with a gradient background and continuous rotation effect. When you hover over the card, the text color should transition to full opacity.

Congratulations! You have created a magic-themed animated card using CSS. Feel free to modify the styles further to customize the appearance and bring your own magical touch to the card.

That concludes this tutorial. Enjoy creating more enchanting designs using CSS!
