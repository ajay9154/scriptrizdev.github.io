Creating Social Media Icons Using HTML and CSS

By
wynkCode -
May 9, 2023

"Creating social media icons using HTML and CSS allows you to incorporate attractive and interactive icons representing various social media platforms into your website. With HTML, you can structure the elements of your webpage, while CSS provides the styling and visual enhancements.

In this tutorial, you will learn how to design and implement social media icons using HTML and CSS. We will cover the process of creating HTML markup for the icons and then applying CSS styles to customize their appearance, including colors, hover effects, and positioning.

By following this tutorial, you will gain the skills to enhance your website's design by adding eye-catching social media icons that link to your social media profiles. Whether you're a beginner or have some experience with HTML and CSS, this tutorial will provide you with the knowledge and guidance to create professional-looking social media icons for your website."

html
<!DOCTYPE html>
                  <html lang="en">
                  
                  <head>
                      <meta charset="UTF-8">
                      <meta http-equiv="X-UA-Compatible" content="IE=edge">
                      <meta name="viewport" content="width=device-width, initial-scale=1.0">
                      <title>wynkCode - Social Media Icons</title>
                      <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.3.0/css/all.min.css"
                          integrity="sha512-SzlrxWUlpfuzQ+pcUCosxcglQRNAq/DZjVsC0lE40xsADsfeQoEypE+enwcOiGjk/bSuGGKHEyjSoQ1zVisanQ=="
                          crossorigin="anonymous" referrerpolicy="no-referrer" />
                  
                      <link rel="stylesheet" href="./style.css">
                  
                  </head>
                  
                  <body>
                      <div class="social-menu">
                          <ul>
                              <li><a href="" target="_blank"><i class="fa-brands fa-youtube"></i></a></li>
                              <li><a href="" target="_blank"><i class="fa-brands fa-instagram"></i></a></li>
                              <li><a href="" target="_blank"><i class="fa-brands fa-linkedin-in"></i></a></li>
                              <li><a href="" target="_blank"><i class="fa-brands fa-github"></i></a></li>
                          </ul>
                      </div>
                  </body>
                  
                  </html>

You can copy and paste this code into your code editor to apply the specified CSS styles for the social media icons.

css
* {
                  margin: 0;
                  padding: 0;
                  box-sizing: border-box;
              }
              
              body {
                  background: #d8d8d8;
              }
              
              .social-menu ul {
                  position: absolute;
                  top: 50%;
                  left: 50%;
                  padding: 0;
                  margin: 0;
                  transform: translate(-50%, -50%);
                  display: flex;
              }
              
              .social-menu ul li {
                  list-style: none;
                  margin: 0 15px;
              }
              
              .social-menu ul li a .fa-brands {
                  font-size: 30px;
                  line-height: 60px;
                  transition: .3s;
                  color: #000;
              }
              
              .social-menu ul li .fa-brands:hover {
                  color: #fff;
              }
              
              .social-menu ul li a {
                  text-decoration: none;
              }
              
              .social-menu ul li a i {
                  position: relative;
                  display: block;
                  width: 60px;
                  height: 60px;
                  border-radius: 50%;
                  background-color: #fff;
                  text-align: center;
                  transition: .6s;
                  box-shadow: 0 5px 4px rgba(0,0,0,.5);
              }
              
              .social-menu ul li a i:hover {
                  transform: translate(0, -10%);
              }
              
              .social-menu ul li:nth-child(1) a i:hover {
                  background-color: rgba(190, 14, 14, 0.829);
              }

                  background-color: #e44084;
              }
              
              .social-menu ul li a i:hover {
                  color: white;
              }
              
              .social-menu ul li:nth-child(3) a i:hover {
                  background-color: #0077b5;
